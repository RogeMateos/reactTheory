## ğŸš€ Getting Started

### Prerequisites
- Node.js installed on your machine
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/RogeMateos/reactTheory.git
cd reactTheory
```

2. Install dependencies:

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to the local server URL (typically `http://localhost:3000`)
