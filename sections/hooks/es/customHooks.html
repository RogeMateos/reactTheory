Custom Hooks: Reutilizar lГіgica en React

ВҝQuГ© es un custom hook?

Los custom hooks son una de las caracterГӯsticas mГЎs poderosas de React. Te permiten extraer lГіgica de componentes y reutilizarla en mГәltiples lugares de tu aplicaciГіn, mejorando la organizaciГіn, mantenibilidad y composiciГіn del cГіdigo.

Un custom hook es una funciГіn de JavaScript que:

  • Su nombre empieza con use (convenciГіn obligatoria)
  • Puede usar otros hooks de React dentro
  • Encapsula lГіgica que quieres reutilizar
  • Retorna valores que otros componentes pueden usar
// Ejemplo bГЎsico de un custom hook
function useWindowWidth() {
  const [width, setWidth] = useState(window.innerWidth)

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth)
    window.addEventListener('resize', handleResize)
    return () => window.removeEventListener('resize', handleResize)
  }, [])

  return width
}

ВҝPor quГ© crear custom hooks?

Problema: LГіgica duplicada

Imagina que tienes varios componentes que necesitan detectar el ancho de la ventana:

  • CГіdigo duplicado en mГәltiples componentes
  • Si necesitas cambiar la lГіgica, debes actualizarla en todos lados

Vamos a transformar la lГіgica de un router casero en un custom hook reutilizable.

Antes: Componente con lГіgica mezclada

Problemas:

  • El componente App mezcla lГіgica de routing con UI
  • No podemos reutilizar esta lГіgica en otros componentes
  • DifГӯcil de mantener y testear

Paso 1: Extraer la lГіgica a un custom hook

// hooks/useRouter.js
import { useState, useEffect } from 'react'

export function useRouter() {
  // Estado para la ruta actual
  const [currentPath, setCurrentPath] = useState(window.location.pathname)

  // Efecto para detectar cambios de navegaciГіn
  useEffect(() => {
    const handleLocationChange = () => {
      setCurrentPath(window.location.pathname)
    }

    // Escuchar cambios en el historial (botГіn atrГЎs/adelante)
    window.addEventListener('popstate', handleLocationChange)

    // Cleanup: remover el listener al desmontar
    return () => {
      window.removeEventListener('popstate', handleLocationChange)
    }
  }, [])

  // FunciГіn para navegar programГЎticamente
  const navigateTo = (path) => {
    window.history.pushState({}, '', path)
    setCurrentPath(path)
  }

  // Retornar la API del hook
  return {
    currentPath,
    navigateTo,
  }
}

ComposiciГіn de hooks

Una de las mayores ventajas de los custom hooks es que pueden usar otros hooks, incluidos otros custom hooks:

// Hook base: useLocalStorage
function useLocalStorage(key, initialValue) {
  const [value, setValue] = useState(() => {
    const item = window.localStorage.getItem(key)
    return item ? JSON.parse(item) : initialValue
  })

  useEffect(() => {
    window.localStorage.setItem(key, JSON.stringify(value))
  }, [key, value])

  return [value, setValue]

// Hook compuesto: useDarkMode (usa useLocalStorage)
function useDarkMode() {
  const [isDark, setIsDark] = useLocalStorage('darkMode', false)

  useEffect(() => {
    if (isDark) {
      document.body.classList.add('dark')
    } else {
      document.body.classList.remove('dark')
    }
  }, [isDark])

  const toggle = () => setIsDark((prev) => !prev)

  return { isDark, toggle }
}

Reglas de los custom hooks

1. El nombre DEBE empezar con "use"

// вң… Bien
function useCounter() {}
function useAuth() {}
function useApi() {}

// вқҢ Mal
function counter() {}
function getAuth() {}
function apiHook() {}

ВҝPor quГ©? React necesita identificar los hooks para aplicar sus reglas internas.

2. Solo llamar hooks en el nivel superior

No puedes llamar hooks dentro de condiciones, loops o funciones anidadas:

// вқҢ Mal
function MyComponent() {
  if (condition) {
    const value = useMyHook() // ВЎError!
  }

  for (let i = 0; i < 10; i++) {
    useEffect(() => {}) // ВЎError!
  }
}

// вң… Bien
function MyComponent() {
  const value = useMyHook()

  useEffect(() => {
    if (condition) {
      // LГіgica condicional dentro del efecto
    }
  }, [condition])
}

3. Solo llamar hooks desde componentes o custom hooks

// вң… Bien: Desde un componente
function MyComponent() {
  const value = useMyHook()
}

// вң… Bien: Desde otro custom hook
function useCustomHook() {
  const value = useMyHook()
}

// вқҢ Mal: Desde una funciГіn normal
function helperFunction() {
  const value = useMyHook() // ВЎError!
}

CuГЎndo crear un custom hook

Crea un custom hook cuando:

  • вң… Tienes lГіgica duplicada en mГәltiples componentes
  • вң… La lГіgica usa hooks de React
  • вң… Quieres separar lГіgica de UI para mejor organizaciГіn
  • вң… Necesitas testear la lГіgica de forma aislada
  • вң… Quieres compartir lГіgica entre componentes

NO crees un custom hook si:

  • вқҢ Solo usas la lГіgica en un componente
  • вқҢ La lГіgica es muy simple (una lГӯnea)
  • вқҢ No usa ningГәn hook de React
  • вқҢ Es solo una funciГіn helper normal

Ejemplo 1: useLocalStorage

Un hook para sincronizar estado con localStorage:

import { useState, useEffect } from 'react'

export function useLocalStorage(key, initialValue) {
  // Obtener valor inicial del localStorage
  const [value, setValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key)
      return item ? JSON.parse(item) : initialValue
    } catch (error) {
      console.error(error)
      return initialValue
    }
  })

  // Actualizar localStorage cuando cambia el valor
  useEffect(() => {
    try {
      window.localStorage.setItem(key, JSON.stringify(value))
    } catch (error) {
      console.error(error)
    }
  }, [key, value])

  return [value, setValue]
}

// Uso:
function App() {
  const [name, setName] = useLocalStorage('userName', 'AnГіnimo')

  return (
    <div>
      <input value={name} onChange={(e) => setName(e.target.value)} />
      <p>Hola, {name}!</p>
    </div>
  )
}

Ejemplo 2: useFetch

Un hook para hacer peticiones HTTP:

import { useState, useEffect } from 'react'

export function useFetch(url) {
  const [data, setData] = useState(null)
  const [loading, setLoading] = useState(true)
  const [error, setError] = useState(null)

  useEffect(() => {
    setLoading(true)
    setError(null)

    fetch(url)
      .then((response) => {
        if (!response.ok) throw new Error('Error en la peticiГіn')
        return response.json()
      })
      .then((data) => {
        setData(data)
        setLoading(false)
      })
      .catch((error) => {
        setError(error.message)
        setLoading(false)
      })
  }, [url])

  return { data, loading, error }
}

// Uso:
function UserProfile({ userId }) {
  const { data: user, loading, error } = useFetch(`/api/users/${userId}`)

  if (loading) return <div>Cargando...</div>
  if (error) return <div>Error: {error}</div>

  return (
    <div>
      <h2>{user.name}</h2>
      <p>{user.email}</p>
    </div>
  )
}

Ejemplo 6: useDebounce

Un hook para retrasar la actualizaciГіn de un valor:

import { useState, useEffect } from 'react'

export function useDebounce(value, delay = 500) {
  const [debouncedValue, setDebouncedValue] = useState(value)

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value)
    }, delay)

    return () => {
      clearTimeout(timer)
    }
  }, [value, delay])

  return debouncedValue
}

// Uso:
function SearchBar() {
  const [searchTerm, setSearchTerm] = useState('')
  const debouncedSearchTerm = useDebounce(searchTerm, 500)

  useEffect(() => {
    if (debouncedSearchTerm) {
      // Hacer bГәsqueda en API
      console.log('Buscando:', debouncedSearchTerm)
    }
  }, [debouncedSearchTerm])

  return (
    <input
      type="text"
      value={searchTerm}
      onChange={(e) => setSearchTerm(e.target.value)}
      placeholder="Buscar..."
    />
  )
}

Ejercicio 1: Crear useCounter

DesafГӯo: Crea un hook para manejar un contador con increment, decrement y reset.

import { useState } from 'react'

export function useCounter(initialValue = 0) {
  const [count, setCount] = useState(initialValue)

  const increment = () => setCount((prev) => prev + 1)

    count,
    increment,
    decrement,
    reset,
  }
}

    </div>
  )
}

import { useState, useEffect } from 'react'

export function useOnlineStatus() {
  const [isOnline, setIsOnline] = useState(navigator.onLine)

  useEffect(() => {
    const handleOnline = () => setIsOnline(true)

      window.removeEventListener('online', handleOnline)
      window.removeEventListener('offline', handleOffline)

  return isOnline
}

Ejercicio 5: Crear useClickOutside

DesafГӯo: Crea un hook que detecte clicks fuera de un elemento.

SoluciГіn:

import { useEffect, useRef } from 'react'

export function useClickOutside(callback) {
  const ref = useRef(null)

  useEffect(() => {
    const handleClick = (event) => {
      if (ref.current && !ref.current.contains(event.target)) {
        callback()
      }
    }

    document.addEventListener('mousedown', handleClick)

    return () => {
      document.removeEventListener('mousedown', handleClick)
    }
  }, [callback])

  return ref
}

// Uso:
function Dropdown() {
  const [isOpen, setIsOpen] = useState(false)
  const dropdownRef = useClickOutside(() => setIsOpen(false))

  return (
    <div ref={dropdownRef}>
      <button onClick={() => setIsOpen(!isOpen)}>
        Toggle
      </button>

      {isOpen && (
        <div className="dropdown-menu">
          <p>Contenido del dropdown</p>
        </div>
      )}
    </div>
  )
}

DesafГӯo: Crea un hook useTodos que combine useLocalStorage y useToggle.

SoluciГіn:

import { useLocalStorage } from './useLocalStorage'

export function useTodos() {
  const [todos, setTodos] = useLocalStorage('todos', [])

  const addTodo = (text) => {
    const newTodo = {
      id: Date.now(),
      text,
      completed: false,
    }
    setTodos([...todos, newTodo])
  }

  const toggleTodo = (id) => {
    setTodos(
      todos.map((todo) =>
        todo.id === id ? { ...todo, completed: !todo.completed } : todo
      )
    )
  }

  const removeTodo = (id) => {
    setTodos(todos.filter((todo) => todo.id !== id))
  }

  return {
    todos,
    addTodo,
    toggleTodo,
    removeTodo,
  }
}

// Uso:
function TodoApp() {
  const { todos, addTodo, toggleTodo, removeTodo } = useTodos()
  const [input, setInput] = useState('')

  const handleSubmit = (e) => {
    e.preventDefault()
    if (input.trim()) {
      addTodo(input)
      setInput('')
    }
  }

  return (
    <div>
      <form onSubmit={handleSubmit}>
        <input
          value={input}
          onChange={(e) => setInput(e.target.value)}
          placeholder="Nueva tarea"
        />
        <button type="submit">Agregar</button>
      </form>

      <ul>
        {todos.map((todo) => (
          <li key={todo.id}>
            <input
              type="checkbox"
              checked={todo.completed}
              onChange={() => toggleTodo(todo.id)}
            />
            <span
              style={{
                textDecoration: todo.completed ? 'line-through' : 'none',
              }}
            >
              {todo.text}
            </span>
            <button onClick={() => removeTodo(todo.id)}>
              Eliminar
            </button>
          </li>
        ))}
      </ul>
    </div>
  )
}
