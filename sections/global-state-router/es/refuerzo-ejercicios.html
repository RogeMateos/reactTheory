Ejercicio 1 — Mejorar la página de detalle

La página de detalle "está bastante bien, pero hay que arreglarle los estilos totalmente". El objetivo es que deje de parecer un prototipo y se convierta en una vista que puedas enseñar sin pedir perdón.

Preguntate:

  ¿Qué sobra?
  ¿Qué falta?

Ejercicio 2.1 — Breadcrumbs (opcional pero recomendable)

Que el usuario sepa en qué parte de la aplicación está:

<nav aria-label="Ruta de navegación">
  <ol>
    <li>{/* Enlace a la lista de ofertas */}</li>
    <li aria-current="page">{/* Título del trabajo */}</li>
  </ol>
</nav>
Puntos importantes:

✅ Muestra la jerarquía de navegación
✅ Usá aria-label y aria-current para accesibilidad
✅ El último elemento NO debería ser un enlace

Ejercicio 2.2 — Marcar el enlace activo

Usá la información de la URL para marcar como "activo" el enlace correspondiente del menú.

Necesitás:

1. Obtener la ruta actual
   (pensá en hooks del router que ya usaste)

2. Comparar la ruta actual con la ruta del enlace

3. Aplicar una clase CSS diferente cuando coincidan

4. Añadir aria-current="page" para accesibilidad
// Un patrón común es crear un componente NavLink
// que envuelva tu Link y añada esta lógica

<NavLink
  to="/jobs"
  className={({ isActive }) =>
    isActive ? "nav-link active" : "nav-link"
  }
>
  Empleos
</NavLink>

Ejercicio 4 — Inicializar todos los filtros desde la URL

Los filtros "funcionan, pero cuando recargás, no se están inicializando correctamente". Ya vimos cómo hacerlo con el buscador de texto. Ahora aplicalo al resto.

Para cada filtro (modalidad, contrato, salario...):

Al cambiar el filtro en la UI:
  → Actualizar estado en React
  → Actualizar la URL con el nuevo valor

Al cargar o refrescar:
  → Leer el parámetro de la URL
  → Inicializar el estado del filtro con ese valor
// Custom hook por filtro
function useModeFilter() {
  const [mode, setMode] = useState('')

  // useEffect para leer la URL al montar
  // función para actualizar mode y URL

  return { mode, updateMode }
}

// Función para actualizar un filtro
const updateFilter = (name, value) => {
  const params = new URLSearchParams(window.location.search)

  if (value) params.set(name, value)
  else params.delete(name)

  // actualizar URL
}
// Ejemplo en componente
function Filters() {
  const { mode, updateMode } = useModeFilter()

  return (
    <select value={mode} onChange={(e) => updateMode(e.target.value)}>
      <option value="">Cualquiera</option>
      <option value="remote">Remoto</option>
      <option value="hybrid">Híbrido</option>
      <option value="onsite">Presencial</option>
    </select>
  )
}
