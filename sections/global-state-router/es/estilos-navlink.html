Estilos Activos con NavLink

¿Qué problema resuelve NavLink?

Imaginá que estás en una web de empleo:

🏠 Inicio     💼 Empleos     📞 Contacto

Entrás en Empleos. Pero el menú sigue viéndose igual:

🏠 Inicio     💼 Empleos     📞 Contacto
❌ No sabés visualmente dónde estás
❌ Podés seguir pulsando "Empleos" una y otra vez
❌ Mala experiencia de usuario

La solución: NavLink

NavLink es un Link inteligente. Además de navegar, sabe: "¿Estoy actualmente en esta página?"

Link normal NavLink
Navegar✅✅
Detectar página actual❌✅
isActive❌✅
Estilos activos❌✅
aria-current automático❌✅
Menús de navegación😐⭐

Visualmente

Estamos en Home (URL: /)

🏠 Inicio
💼 Empleos
📞 Contacto

Estamos en Jobs (URL: /jobs)

🏠 Inicio
💼 Empleos  ← ACTIVO
📞 Contacto

Ahora el usuario sabe exactamente dónde está.

¿Cómo sabe NavLink si está activo?

React Router te da una variable llamada isActive:

// Cuando NO estamos en /jobs
isActive = false
// <a class="header-nav-link">Empleos</a>

// Cuando SÍ estamos en /jobs
isActive = true
// <a class="header-nav-link navLinkActive">Empleos</a>

NavLink añade una clase extra navLinkActive cuando la ruta coincide.

El código

<NavLink
  to="/jobs"
  className={({ isActive }) =>
    isActive
      ? "header-nav-link navLinkActive"
      : "header-nav-link"
  }
>
  Empleos
</NavLink>

Usá Link cuando Usá NavLink cuando

Ejemplos: 🏠 Inicio, 💼 Empleos, 📞 Contacto, ⚙️ Config

CSS para el estado activo

.header-nav-link.navLinkActive {
  color: yellow;
  font-weight: bold;
  border-bottom: 2px solid yellow;
}
Sin estilos activos:
🏠 Inicio
💼 Empleos
📞 Contacto

Con estilos activos (en /jobs):

💼 EMPLEOS
──────────
📞 Contacto

Desactivar el enlace actual

Si ya estás en Jobs, ¿para qué volver a pulsar Jobs? Podés bloquear el clic:

.header-nav-link.navLinkActive {
  pointer-events: none;
}
Antes:
💼 Empleos
(click)
(click)  ← redundante

Después:
💼 Empleos (actual)
🚫 No clicable

Bonus oculto: Accesibilidad

Cuando el link está activo, React Router añade automáticamente:

<a
  class="navLinkActive"
  aria-current="page"
>
  Empleos
</a>

Esto ayuda a:

✅ Lectores de pantalla
✅ Navegación por teclado
✅ Accesibilidad

No tenés que hacer nada.
NavLink lo hace por vos.

Referencia rápida

import { NavLink } from 'react-router'

<NavLink
  to="/jobs"
  className={({ isActive }) =>
    isActive
      ? "header-nav-link navLinkActive"
      : "header-nav-link"
  }
>
  Empleos
</NavLink>
Link
│
└── Solo navega

NavLink
│
├── Navega
├── Detecta ruta activa
├── Tiene isActive
├── Añade aria-current
└── Permite estilos visuales
NavLink = Link + "sé dónde está el usuario actualmente". Usalo siempre en menús de navegación.
