Understanding the useEffect Hook in React

The useEffect hook is a fundamental part of React's functional components, allowing you to perform side effects such as data fetching, subscriptions, or manual DOM manipulations. Here's a detailed breakdown of how to use useEffect effectively.

What is useEffect?

Basic Syntax


import React, { useState, useEffect } from 'react';

function ExampleComponent() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    // Side effect: Update the document title
    document.title = `Count: ${count}`;
  }, [count]); // Dependency array

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(prev => prev + 1)}>Add</button>
    </div>
  );
}
